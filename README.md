**System requirements**

* *nix system
* Apache 2.4
* PHP 5.6
* MySQL 5.5

**Web server**

Install app on any directory. Need activate _mod_rewrite_, and set _AllowOverride All_ for directory (_.htaccess_).

**Migration**

Create new database, run SQL from _dump.sql_ file.

**Configuration**

Edit _conf.php_ file.
